Understanding Binary Search

Binary search is one of the most fundamental and efficient searching algorithms in computer science. Unlike linear search which examines every element (O(n) complexity), binary search repeatedly divides the search space in half, achieving logarithmic O(log n) time complexity.

The key requirement is that your data must be sorted. When this condition is met, binary search can find elements in large datasets incredibly quickly‚Äîsearching through a million items requires only about 20 comparisons!

Python
def binary_search(arr, target): # Initialize left and right pointers left = 0 right = len(arr) - 1 while left <= right: # Calculate middle index (avoids overflow) mid = left + (right - left) // 2 # Check if target is at mid if arr[mid] == target: return mid # If target is greater, ignore left half elif arr[mid] < target: left = mid + 1 # If target is smaller, ignore right half else: right = mid - 1 # Target not found return -1
Example Usage:
>>> sorted_array = [1, 3, 5, 7, 9, 11, 13, 15]
>>> binary_search(sorted_array, 7)
3
üí° Key Concepts
  • ‚Ä¢ The array must be sorted before applying binary search
  • ‚Ä¢ Use left + (right - left) // 2 instead of (left + right) // 2 to prevent integer overflow
  • ‚Ä¢ The condition is while left <= right (not just <) to include single element case
  • ‚Ä¢ Time complexity: O(log n) ‚Ä¢ Space complexity: O(1) for iterative version
