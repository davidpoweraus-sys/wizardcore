Check Current Supabase URL

Open browser console (F12) on https://offensivewizard.com

Then paste this code:

// Check what URL the app is actually using
fetch('https://offensivewizard.com/_next/static/chunks/app/layout.js')
  .then(r => r.text())
  .then(text => {
    const matches = text.match(/auth\.offensivewizard\.com|api\/supabase-proxy/g)
    console.log('Found URLs:', matches)
    if (matches?.includes('api/supabase-proxy')) {
      console.log('✅ Using proxy URL - GOOD!')
    } else {
      console.log('❌ Still using old direct URL - Need to rebuild with new env var')
    }
  })

// Or check the environment directly
console.log('Checking Supabase client...')
// This will show what URL is configured

Quick Test

Run this in console on offensivewizard.com:

// Test if proxy endpoint exists
fetch('/api/supabase-proxy/health')
  .then(r => {
    console.log('Proxy status:', r.status)
    return r.text()
  })
  .then(data => console.log('Proxy response:', data))
  .catch(e => console.error('Proxy error:', e))
